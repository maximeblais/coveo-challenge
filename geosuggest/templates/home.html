GeoSuggest

A RESTful API that provides suggestions for cities

Examples:

  • GET /suggestions?q=Sherbrooke (single result)
  • GET /suggestions?q=New (multiple results)
  • GET /suggestions?q=SomeRandomCityInTheMiddleOfNowhere (no result)
  • GET /suggestions?q=New&latitude=39.68372&longitude=-75.74966 (multiple results, exact location)
  • GET /suggestions?q=Londo&latitude=43.70011&longitude=-79.4163 (multiple results, near match)

A little extra:

  • GET /suggestions?q=Sherbrooke&visualize=true (single result)
  • GET /suggestions?q=New&visualize=true (multiple results)
  • GET /suggestions?q=New&latitude=39.68372&longitude=-75.74966&visualize=true (multiple results, exact location)
  • GET /suggestions?q=Londo&latitude=43.70011&longitude=-79.4163&visualize=true (multiple results, near match)
